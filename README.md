## Project Description
A Profile Management System built with HTML, Servlet, JavaBean, JDBC, and JSP
following MVC. Stores and manages student profiles in a Derby database (StudentProfilesDB).

## Features Implemented
- Add profile (form → Servlet → JDBC → database)
- View all profiles (Bootstrap table)
- Search profile by Student ID or Name

## How to Run
1. Run database.sql in NetBeans Services to create the Profile table.
2. Open project in NetBeans IDE 8.2, run on GlassFish.
3. Access via index.html.
